Assignment #1




Assignment #1 was to setup our work environment.
I completed the following tasks:
  1. Complete Course Registration
    • First I completed the registration Google form
    • Then I joined the Piazza forum and checked it out over the next few days.
    • Finally I did the pre-course survey.
  2. Get a web server up and running
    • I didn't bother with XAMP. I just ran "sudo apt-get install apache2" on my Linux machine
    • Then I spent a while configuring Apache. I made a public_html folder and got stuff to display on local host.
    • Then I saw a link on Piazza about Python's simple server. And it was way easier to use.
    • So now I just run "python -m SimpleHTTPServer" in a second terminal.
  3. Install Netbeans
    • I personally dislike IDEs.
    • Since they aren't required for this class, I just didn't install any.
    • I typically use either gedit with code highlighting on, and some other things like line numbers.
    • After I configured Sublime 3 though I began to really like it. So I now use that for the most part.
